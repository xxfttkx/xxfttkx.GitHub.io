从 Flutter 3 / Material 3 开始：

```
ThemeData(
  useMaterial3: true,
)
```

如果你 **没有显式指定 `colorScheme` 或 `primaryColor`**，Flutter 会：

* 自动用 **默认种子色（偏紫）**
* 生成整套 `ColorScheme`
* 所有 `FloatingActionButton / IconButton / Focus / Selection`
  都会吃这个紫色

我喜欢绿色😡

为了配合theme，该多使用colorScheme吧，别再写死颜色了。
