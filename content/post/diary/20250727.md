听说**2023 年后 GitHub 移除了全局自动 watch 自己仓库的功能**。可以`gh repo list <你的用户名> --limit 1000 --json nameWithOwner -q '.[].nameWithOwner' | xargs -n 1 gh repo watch`。使用`gh CLI`结果被告知`gh CLI 的新版本已经移除了 gh repo watch 命令。`，所以要使用`gh api`。
