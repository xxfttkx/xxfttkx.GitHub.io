> 怎么查询windows使用的字体

VS Code 实际是基于 Chromium 内核，你可以这样查：

1. 在 VS Code 中按下 `F1`
2. 输入 `Developer: Toggle Developer Tools` → 回车
3. 在打开的开发者工具中，点击上方的 **Elements** 标签
4. 找一段你写的中文代码（例如“你好世界”）
5. 点击它，对应右边 `Styles` → 找到 `font-family`，可以看到：

Segoe WPC,Segoe UI,sans-serif
